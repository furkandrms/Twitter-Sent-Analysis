
# Twitter Sentiment Analysis
```

Twitter Sentiment Analysis, sosyal medya platformu Twitter üzerindeki metin tabanlı içeriklerin duygu analizini yapabilen bir projedir. Bu proje, Python programlama dili ve çeşitli kütüphaneler kullanılarak geliştirilmiştir.

## Projenin Amacı

Twitter Sentiment Analysis projesi, Twitter üzerindeki tweetlerin duygu durumunu analiz etmeyi hedeflemektedir. Kullanıcılar, belirli bir Twitter hesabının son tweetlerini girerek bu tweetlerin pozitif, negatif veya nötr duygular içerip içermediğini öğrenebilirler. Bu analiz, kullanıcılara bir hesabın genel duygu durumunu anlamalarına yardımcı olur.

## Kullanılan Teknolojiler

Proje aşağıdaki teknolojileri kullanmaktadır:

- Python: Proje, Python programlama dilinde geliştirilmiştir.
- Tweepy: Twitter API entegrasyonu için Tweepy kütüphanesi kullanılmıştır.
- Sentiment Analysis: Tweetlerin duygu analizi için önceden eğitilmiş bir duygu analizi modeli kullanılmıştır.
- Streamlit: Web arayüzü, Streamlit kütüphanesi kullanılarak oluşturulmuştur.

## Nasıl Kullanılır?

Proje kullanımı oldukça basittir. Aşağıdaki adımları takip ederek projeyi çalıştırabilirsiniz:

1. Projenin GitHub deposunu klonlayın:
   ```
   git clone https://github.com/furkandrms/twitter-sentiment-analysis.git
   ```

2. Terminali açın ve projenin dizinine gidin:
   ```
   cd twitter-sentiment-analysis
   ```

3. Gerekli bağımlılıkları yüklemek için aşağıdaki komutu çalıştırın:
   ```
   pip install -r requirements.txt
   ```

4. Twitter API kimlik bilgilerinizi `get_tweets.py` dosyasında ilgili alanlara girin.

5. Uygulamayı çalıştırın:
   ```
   streamlit run twitter_sent.py
   ```

6. Web tarayıcınızda `http://localhost:8501` adresine gidin ve Twitter Sentiment Analysis uygulamasını kullanmaya başlayın.
```
